# sistema de gestion de incidentes

## descripcion
aplicacion en typescript para gestionar incidentes tecnologicos. permite crear, listar y actualizar el estado de incidentes.

## requisitos
- node.js
- pnpm

## instalacion

```bash
# clonar repositorio
git clone <url-del-repositorio>

# instalar dependencias
pnpm install

# ejecutar en modo desarrollo
pnpm dev

# compilar a javascript
pnpm build

# ejecutar version compilada
pnpm start
